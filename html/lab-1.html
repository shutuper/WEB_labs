Опис предметного середовища:

Як основну предметну область ми обрали вузькоспеціалізовану, при тому не менш важливу галузь – продаж виробів із різнокольорового синтетичного волосся. Дана область не має широкої популярності, тому часто замовчується, вважається дещо інтимною. Ми ставимо за мету популяризувати перуки, як засіб самовираження та створити зручний та сучасний сайт для їх продажу.

Опис бізнес-логіки: Головна ідея даного WEB-додатку - це продаж перук. Для забезпечення зручного інтерфейсу сайт містить таблицю з наявним товаром, візуалізовану фотографіями. Справа знаходиться форма, де ввівши свої дані, користувач може замовити товар. Внизу знаходиться кнопка, натиснувши яку, користувач буде перенаправлений на сторінку з описом предметної області. З цієї сторінки користувач може повернутись на початкову сторінку.

Опис функціоналу на користувачів:

1. Перегляд списку товарів
2. Заповнення форми
3. Замовлення товару
4. Авторизація
5. Добавлення товару в список улюблених
6. Добавлення товару в кошик
7. Перегляд сторінки з описом предметної області

Опис функціоналу для адміна:

1. Авторизація
2. Доступ до всіх сторінок сайта
3. Додавання товарів
4. Редагування вже існуючих товарів
5. Видалення товарів
6. Перегляд транзакцій
7. Перегляд популярності(кількість доданих в список улюблених) товарів

HTML-код посилання

                  
    <div class="btn-group">
      <a href="program_subject.html" class="btn btn-primary border mt-2">Cторінку с описом предметної галузі</a>
      <a href="D:\Atom\WEB_labs\html\lab-1.html" class="btn btn-primary border mt-2">Висновок</a>
    </div>
Для створення посилання необхідно повідомити браузеру, що є посиланням, а також вказати адресу документа, на який слід зробити посилання. Обидві дії виконуються за допомогою тега <a>. <a href="URL"> текст посилання </a> Атрибут href визначає URL (Universal Resource Locator, універсальний покажчик ресурсу), іншими словами, адресу документа, на який слід перейти, а вміст контейнера <a> є посиланням. Текст, розташований між тегами <a> і </a>, за замовчуванням стає синього кольору і підкреслюється.

<table class="table table-hover table-dark">

Для створення таблиці використовується паринй тег <table> </table> який створює таблицю в HTML документі. Таблиця має складатись хочаб з одног рядка <tr> та однієї комірки (<th> - комірка-заголовок або <td> - комірка таблиці) Таблиця, створена тегом <table> може бути розділена на верхній, нижній колонтитули та основну частину за допомогою тегів <thead>, <tfoot>, <tbody>. До таблиці можна додати підпис, використовуючи тег <caption>. Усередині таблицю <table> можна ділити не тільки по рядках, але і по колонках використовуючи теги <col> і <colgroup>

HTML-код форми 

                    
    <form>
      <div class="mb-3">
        <label for="product" class="form-label">Товар:</label>
        <input type="text" class="form-control" id="product" aria-describedby="emailProduct">
        <div id="emailHelp" class="form-text">Tap on product you want to buy.</div>
      </div>
      <div class="mb-3">
        <label for="quantity" class="form-label">Кількість:</label>
        <input type="number" min="0" class="form-control" id="quantity">
      </div>
      <div class="mb-3">
        <label for="firstName" class="form-label">Ім'я:</label>
        <input type="text" class="form-control" id="firstName">
      </div>
      <div class="mb-3">
        <label for="lastName" class="form-label">Прізвище:</label>
        <input type="text" class="form-control" id="lastName">
      </div>
      <div class="mb-3">
        <label for="numberOfTelephone" class="form-label">Телефон:</label>
        <input type="tel" class="form-control" id="numberOfTelephone">
    </div>
      <div class="mb-3">
        <label for="postOffice" class="form-label">Місто, номер відділення НП:</label>
        <input type="text" class="form-control" id="postOffice">
      </div>
      <button type="submit" class="btn btn-primary">Продовжити</button>
    </form>
<Form> є блоковим елементом, який визначає інтерактивну частину веб-сторінки. Як результат, всі елементи управління (такі як <input>, <textarea> або <button>) повинні знаходитися всередині елемента <form>. Два атрибути HTML необхідні: action містить адресу, який визначає, куди буде відправлена інформація форми; method може бути або GET, або POST і визначає, як буде відправлена інформація форми.

HTML-код зображення 

                  
    <div class="col-6">
      <img class="img img-responsive w-100 border rounded" src="../img/kosobyc.jpg" alt="">
    </div>
    <div class="col-6">
      <img class="img img-responsive w-100 border rounded" src="https://mandarinonline.ru/images/stories/virtuemart/product/%D0%9F%D0%9A162,2.jpg" alt="">
    </div>
Тег <img> використовується для вставки графічного зображення (картинки) в HTML документ. HTML тег <img> має два обов'язкових атрибути: src - адреса файлу картинки і alt - альтернативний текст, який буде відображений, якщо картинка не може бути завантажена. Також не зайвим буде використання глобального атрибута title. Зображення з тега <img> може бути посиланням. Для цього потрібно обгорнути тег <img> тегом посилання <a>
